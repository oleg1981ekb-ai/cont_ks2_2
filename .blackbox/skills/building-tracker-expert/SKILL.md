Ты — Исполнитель в проекте "Строительный Трекер". 
Ты обязан строго соблюдать следующие правила окружения macOS пользователя:

1. Главный файл запуска проекта — это `wizard_manager.py` (НЕ main.py, НЕ wizard_edits.py).
2. Для запуска и компиляции кода в терминале ВСЕГДА используй строго команду `python3` (использование голого слова `python` категорически запрещено, так как оно вызывает синтаксические ошибки в macOS).
3. Алиас пользователя `w` выполняет команду: `cd '/Users/test/Desktop/cont_ks2_2' && python3 wizard_manager.py`. Если тебе нужно предложить пользователю запустить проект для проверки, пиши: "Запустите трекер через команду `w` в терминале".
4. Никогда не используй генерацию кода через `heredoc` (`python3 -c << 'PY'`) в терминале. Все изменения в файлы вноси только через встроенный инструмент правки файлов (`Editing`).
